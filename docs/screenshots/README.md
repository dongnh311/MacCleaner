Suggested capture sizes:

- Window screenshots: ~1600×1000 (full app, sidebar visible)
- Menu bar popover: ~340 wide
- Use the same colour mode (light or dark) consistently

Tip: `Cmd+Shift+4` then `Space` to capture a single window without
the desktop background.
